# Mixin 代码生成器

## 概述

Mixin 代码生成器将 Mixin 方法直接生成到目标类型中，用户看起来就是原生 Go 代码，没有额外学习成本。

## 使用方式

### 1. 定义 Mixin 方法

在 `internal/mixingen/generator.go` 中定义 Mixin 方法：

```go
var MiddlewareMixinMethods = []MethodDef{
    {
        Name:    "UseCORS",
        Params:  "",
        Results: "*WebApplication",
        Comment: "UseCORS enables CORS middleware",
        Body: `	cfg := a.services.GetService(reflect.TypeOf("cors"))
	if mw := createCORSMiddleware(cfg); mw != nil {
		a.middlewares = append(a.middlewares, mw)
	}
	return a`,
    },
    // ... 更多方法
}
```

### 2. 运行代码生成

```bash
cd core
go run ../internal/mixingen/cmd/main.go
```

或使用 `go generate`:

```bash
go generate ./...
```

### 3. 生成的代码

生成器会在 `core/mixin_generated.go` 中生成代码：

```go
// Code generated by mixingen. DO NOT EDIT.

package core

import "reflect"

// UseCORS enables CORS middleware
func (a *WebApplication) UseCORS() *WebApplication {
    cfg := a.services.GetService(reflect.TypeOf("cors"))
    if mw := createCORSMiddleware(cfg); mw != nil {
        a.middlewares = append(a.middlewares, mw)
    }
    return a
}
```

### 4. 用户使用

```go
builder := core.NewWebApplicationBuilder()

// 注入配置
builder.Services().AddCORS(config)
builder.Services().AddRecovery(nil)

// 构建应用
app := builder.Build()

// 直接使用原生方法
webApp := app.(*core.WebApplication)
webApp.UseCORS().
    UseRecovery().
    UseLogging()

// 正常使用其他方法
app.MapGet("/", handler)
app.Listen(":8080")
```

## 优势

| 特性 | 说明 |
|------|------|
| **零运行时开销** | 编译时生成，无反射 |
| **类型安全** | 编译时检查 |
| **用户友好** | 看起来像原生代码 |
| **可扩展** | 通过定义新的 Mixin 方法扩展功能 |
| **IDE 友好** | 代码补全、跳转正常工作 |


## 扩展 Mixin

要添加新的 Mixin 方法：

1. 在 `generator.go` 中添加方法定义
2. 运行 `go generate ./...`
3. 生成的代码会自动包含新方法

## 与 Wire 集成

Mixin 代码生成器可以与 Wire 结合使用：

1. Wire 负责依赖注入
2. Mixin 生成器负责方法扩展
3. 两者互补，共同提供编译时代码生成能力
